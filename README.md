# Virtual-Mouse-using-Hand-Gesture-Recognition
🚀 Overview
This project enables users to control the mouse using hand gestures captured via a webcam. It uses computer vision techniques to detect hand movements and perform actions like cursor movement, clicking, and scrolling.

🧠 Features
Cursor movement using hand tracking
Left & right click gestures
Scroll functionality
Real-time detection

⚙️ Tech Stack
Python, OpenCV, MediaPipe
